DATE: 2015-04-16

******************* This project is now deprecated *****************

ModbusComm has been renamed ModbusTcp. All future development of the Modbus/TCP
protocol will be done in ModbusTcp.

A new project named ModbusUdp has been created to handle Modbus/UDP transactions.

Please use these other projects for all new development. ModbusComm will be retained
only for backwards compatibility with old projects.
